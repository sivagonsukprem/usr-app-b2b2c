ภาพรวมระบบ TwentyShop

ทำความเข้าใจโครงสร้างและฟีเจอร์หลักของระบบก่อนเริ่มใช้งาน

🏗️ โครงสร้างระบบ B2B Commerce

หน้าหลักของ TwentyShop App — เมนูหลักทั้งหมด
ระบบประกอบด้วย 2 ส่วนหลัก:
1) Admin Web (เปิดในเบราว์เซอร์) — จัดการสินค้า หมวดหมู่ คำสั่งซื้อ โปรโมชั่น ค่าส่ง แชท ข่าวสาร รีวิว และตั้งค่า
2) TwentyShop App (มือถือ Android) — ดูสินค้าโกดัง สั่งซื้อ รับสินค้าเข้า POS ขายหน้าร้าน รายงาน และโปรไฟล์

ขั้นตอนที่ 2 — เข้าสู่ระบบ

เข้าใช้งานได้ทั้ง Admin Web (เบราว์เซอร์) และ TwentyShop App (มือถือ Android)

🌐 Admin Web — เข้าสู่ระบบหลังบ้าน

หน้า Login Admin Web — กรอก Email และ Password แล้วกด "เข้าสู่ระบบ"
วิธีเข้าสู่ระบบ Admin Web:
1. เปิดเบราว์เซอร์ (Chrome แนะนำ) ไปที่ URL ของระบบ
2. กรอก Email และ Password ของบัญชีร้านค้า
3. กดปุ่ม "เข้าสู่ระบบ"

📱 TwentyShop App — เข้าสู่ระบบมือถือ

หน้า Login ของ App — ใส่ Email + Password แล้วกด "เข้าสู่ระบบ"
หน้าแรกหลัง Login สำเร็จ — แถบเมนูด้านล่างสำหรับนำทาง
หมายเหตุ: ใช้อีเมลและรหัสผ่านเดียวกับ Admin Web ระบบแสดงเมนูตามสิทธิ์ของแต่ละ role อัตโนมัติ

ขั้นตอนที่ 3 — จัดการสินค้า

เพิ่ม แก้ไขสินค้า ดู Stock Badge และ Stock Card บัญชีสินค้าในร้าน

🌐 Admin Web — รายการสินค้า

หน้าสินค้าใน Admin Web — รายการสินค้าทั้งหมดของร้าน
เพิ่มหรือแก้ไขสินค้า — ใส่ชื่อ ราคา รูปภาพ และรายละเอียดสินค้า

📱 App — สินค้าในร้านพร้อม Stock Badge

แท็บ "จัดการ" → "จัดการสินค้า" — เพิ่ม/แก้ไขสินค้าในร้าน
แท็บ "สินค้าในร้าน" — badge สี: รับเข้า / ขายแล้ว / คงเหลือ ใต้ราคาสินค้า
Stock Card — กดที่สินค้าเพื่อดูประวัติรับเข้า/ขายออก และยอดคงเหลือแบบบัญชี
Stock Badge ใต้ราคาสินค้า: รับเข้า ขายแล้ว คงเหลือ
กดที่สินค้าเพื่อเปิด Stock Card ดูรายการธุรกรรมทั้งหมดพร้อม running balance

ขั้นตอนที่ 4 — หมวดหมู่สินค้า

สร้างและจัดการหมวดหมู่เพื่อให้ลูกค้าค้นหาสินค้าได้ง่ายขึ้น

🌐 Admin Web — จัดการหมวดหมู่

หน้าหมวดหมู่ — รายการหมวดหมู่สินค้าทั้งหมด
เพิ่ม/แก้ไขหมวดหมู่ — ตั้งชื่อและเลือกไอคอน กดบันทึก
คำแนะนำ: จัดหมวดหมู่ให้ชัดเจน เช่น "เครื่องเขียน", "ของใช้ในบ้าน", "ของเล่น" เพื่อให้ลูกค้าค้นหาสินค้าได้รวดเร็ว

ขั้นตอนที่ 5 — สั่งซื้อสินค้าจากโกดัง

สั่งซื้อสต็อกสินค้าจากโกดังกลางผ่าน TwentyShop App — ตั้งแต่เลือกสินค้า รับคูปอง จนถึงยืนยันออเดอร์

🏷️ ขั้นตอน 1 — ดูสินค้าโกดังและรับคูปอง

หน้ารายละเอียดสินค้า — กด "กดรับคูปองลด ฿10" เพื่อบันทึกคูปองส่วนลด
รับคูปองสำเร็จทั้ง 2 ใบ — จะถูกนำไปใช้อัตโนมัติตอน Checkout

🛒 ขั้นตอน 2 — ตะกร้าสินค้า

ตะกร้าสินค้า — แสดงรายการที่เลือกทั้งหมด 6 รายการ ยอดรวม ฿1,044
เพิ่มสินค้าลงตะกร้า: กดที่สินค้าจากหน้ารายการโกดัง → เลือกจำนวน → กด "เพิ่มลงตะกร้า"
กดไอคอนตะกร้าที่แถบด้านล่างเพื่อดูรายการที่เลือกไว้

💳 ขั้นตอน 3 — เลือกวิธีชำระเงิน

เลือกวิธีชำระ: เงินสด / พร้อมเพย์ / โอนเงิน — กดเลือกแล้วกด "ถัดไป"

📋 ขั้นตอน 4 — ยืนยันคำสั่งซื้อ

หน้ายืนยันออเดอร์ — กรอกที่อยู่จัดส่งก่อน ไม่เช่นนั้นกดสั่งไม่ได้
กรอกที่อยู่ครบแล้ว — กด "ยืนยันคำสั่งซื้อ" เพื่อส่งออเดอร์

✅ ขั้นตอน 5 — สั่งซื้อสำเร็จ และส่ง Slip

สั่งซื้อสำเร็จ! — ระบบออกเลข Order ให้อัตโนมัติ
กด "ส่ง Slip การชำระเงิน" เพื่อแนบหลักฐานโอนเงินให้โกดัง
ออเดอร์ถูกส่งไปยังโกดังแล้ว! รอทีมโกดังตรวจสอบ Slip และอนุมัติออเดอร์ของคุณ

ขั้นตอนที่ 6 — อนุมัติออเดอร์ (ฝั่งโกดัง)

หลังร้านค้าสั่งซื้อ ทีมโกดังตรวจสอบ Slip และอนุมัติออเดอร์ผ่าน Admin Web

🌐 Admin Web — รายการคำสั่งซื้อที่เข้ามา

หน้า "คำสั่งซื้อ" ใน Admin Web — รายการออเดอร์ทั้งหมด พร้อมสถานะแต่ละรายการ
กด "ดูรายละเอียด" — ตรวจสอบรายการสินค้า ที่อยู่ วิธีชำระ และ Slip ที่แนบมา

✅ อนุมัติออเดอร์ → เปลี่ยนสถานะ

กด "อนุมัติ" → สถานะเปลี่ยนเป็น "กำลังเตรียม" — ระบบแจ้งร้านค้าอัตโนมัติ
รายการออเดอร์ทั้งหมด — แสดงสถานะ "กำลังเตรียม" และ "ส่งถึงแล้ว" ชัดเจน

📄 ใบส่งสินค้า / Delivery Note

ใบส่งสินค้า (Delivery Note) — พิมพ์ได้จาก Admin Web กด "พิมพ์บิล" เพื่อแนบไปกับสินค้า
สำหรับร้านค้า: เมื่อโกดังอนุมัติแล้ว จะได้รับ Push Notification แจ้งสถานะออเดอร์ใน App ทันที

ขั้นตอนที่ 7 — เตรียมสินค้า (Checker นับสินค้า)

ทีมโกดังนับและเตรียมสินค้าตามออเดอร์ก่อนจัดส่ง — ผ่านหน้า "เตรียมสินค้า" ใน Admin Web และ App

🌐 Admin Web — หน้าเตรียมสินค้า (กำหนด Checker)

เมนู "เตรียมสินค้า" ใน Admin Web — รายการออเดอร์ที่รอเตรียม พร้อมสถานะการนับ
เลือกออเดอร์ → กำหนด Checker ผู้รับผิดชอบ → เริ่มกระบวนการเตรียมสินค้า

📱 App — Checker นับสินค้าทีละขั้นตอน

หน้าเตรียมสินค้า — ยังไม่ได้เลือก Checker และออเดอร์ (ปุ่มเริ่มยังเป็นสีเทา)
เลือก Checker (ผู้ตรวจ) และเลือกออเดอร์ที่ต้องการเตรียม — ปุ่มเริ่มเปิดใช้งาน
เริ่มนับสินค้า — Progress 0% กรอกจำนวนที่นับได้ในช่องแต่ละรายการ
นับสินค้าคืบหน้า 67% — แถบสีเขียวแสดงความคืบหน้า กรอกรายการที่เหลือต่อ
เมื่อนับครบ 100% กดปุ่ม "ยืนยันเตรียมสินค้า" เพื่อส่งออเดอร์ให้ร้านค้ารับสินค้า

ขั้นตอนที่ 8 — รับสินค้าเข้าร้าน

เมื่อสินค้าจากโกดังมาถึง ตรวจนับ บันทึกรับเข้า พิมพ์รายงาน และแชร์ slip — Stock อัพเดตอัตโนมัติ

📱 App — เข้าหน้ารับสินค้า

แท็บ "จัดการ" → กด "รับสินค้าเข้า" เพื่อเปิดหน้ารับของ
หน้า "รับสินค้าเข้า" — ออเดอร์สถานะ "รอรับเข้า" แสดงที่นี่ กดเพื่อเปิดรายการ

📦 ตรวจนับสินค้าที่รับเข้า

ตรวจนับสินค้า — ตัวเลขสีเขียวหมายถึงสินค้าครบตามออเดอร์ทุกรายการ
พบสินค้าไม่ครบ — ตัวเลขสีแดงพร้อมแจ้ง "ขาด X ชิ้น" ต้องแจ้งโกดัง

⚠️ กรณีสินค้าไม่ครบ — แจ้งโกดังและพิมพ์รายงาน

Dialog "สินค้าไม่ครบ" — เลือก: พิมพ์รายงานขาดส่ง / แชร์ PDF ไปที่ LINE / บันทึกรูปลง Gallery
แนะนำ: กด "พิมพ์รายงานขาดส่ง" แล้ว "แชร์ PDF ไปที่ LINE" เพื่อแจ้งโกดังส่งสินค้าเพิ่มทันที

🖨️ พิมพ์รายงานรับสินค้าและบันทึก

รายงานรับสินค้าเข้า — แสดงรายละเอียดออเดอร์ จำนวนที่นับ และสถานะแต่ละรายการ
เลือก "Save as PDF" เพื่อบันทึกรายงาน หรือพิมพ์ผ่านเครื่องพิมพ์
แชร์ PDF ไปที่ LINE: ส่งรายงานให้โกดังหรือผู้ที่เกี่ยวข้องทันที
บันทึกรูปลง Gallery: เก็บเป็นหลักฐานการรับสินค้าในโทรศัพท์

✅ หลังรับสินค้าเสร็จ — Stock อัพเดตอัตโนมัติ

Admin Web — ออเดอร์ทั้งหมดอัพเดตสถานะ "ส่งถึงแล้ว" แสดงสีเขียว
หน้า "รับสินค้าเข้า" ว่าง — ไม่มีออเดอร์ค้างรอรับอีกแล้ว
Stock อัพเดตอัตโนมัติ! หลังบันทึกรับสินค้า ยอด Stock badge บนหน้าสินค้าในร้านจะอัพเดตทันที

ขั้นตอนที่ 9 — POS ขายหน้าร้าน

ขายสินค้าให้ลูกค้าหน้าร้านโดยตรงผ่าน App — เลือกสินค้า คิดเงิน พิมพ์ใบเสร็จ Thermal Printer

🏪 เปิดหน้า POS และเลือกสินค้า

แท็บ "จัดการ" → กด "POS ขายหน้าร้าน" เพื่อเปิดโหมดขาย
หน้า POS — แสดงสินค้าในร้านทั้งหมด กดสินค้าเพื่อเพิ่มลงรายการขาย ยอดรวมแสดงด้านล่าง
เพิ่มสินค้า: กดที่รูปสินค้า → ระบบเพิ่มลงรายการล่างอัตโนมัติ
ปรับจำนวน: กดปุ่ม − / + ข้างรายการสินค้าด้านล่าง | ล้าง: กดปุ่ม "ล้าง"

💳 เลือกวิธีชำระและยืนยันการขาย

กด "ชำระเงิน" → เลือกวิธีชำระ: เงินสด / โอนเงิน/PromptPay / โอนธนาคาร
ขายสำเร็จ ✓ — แสดงเลขออเดอร์และยอดรวม กด "พิมพ์ใบเสร็จ" เพื่อพิมพ์ทันที

🖨️ ใบเสร็จ Thermal Printer (พิมพ์จริง)

ใบเสร็จ Thermal Printer จริง — ชื่อร้าน เลขออเดอร์ รายการสินค้า และยอดรวม ภาษาไทยชัดเจน
การพิมพ์ใบเสร็จ: ระบบส่งไปยัง RawBT app → พิมพ์ผ่าน Bluetooth Thermal Printer 58mm
รองรับภาษาไทยสมบูรณ์ layout ถูกต้องทุกรายการ

📋 ประวัติออเดอร์ POS และรายละเอียด

หน้า "จัดการออเดอร์" — รายการออเดอร์ POS ทั้งหมด แยกแท็บ: ทั้งหมด / รอดำเนินการ / เสร็จสิ้น / ยกเลิก
กดที่ออเดอร์เพื่อดูรายละเอียด — สินค้า จำนวน ราคา และยอดรวมชัดเจน

ขั้นตอนที่ 10 — โปรโมชั่น / คูปอง

สร้างและจัดการโปรโมชั่น คูปองส่วนลด เพื่อกระตุ้นยอดสั่งซื้อ

🌐 Admin Web — รายการโปรโมชั่น

หน้าโปรโมชั่น — รายการโปรโมชั่นและคูปองทั้งหมด พร้อมสถานะ active/inactive

➕ สร้างคูปองใหม่

สร้างคูปองใหม่ — ตั้งชื่อ มูลค่าส่วนลด ยอดขั้นต่ำ วันหมดอายุ และจำนวนสิทธิ์
รายละเอียดคูปอง — กำหนดเงื่อนไขการใช้งานอย่างละเอียด
วิธีลูกค้าใช้คูปอง: กด "กดรับคูปอง" ในหน้าสินค้า → คูปองถูกเก็บ → นำไปใช้ตอน Checkout อัตโนมัติ

ขั้นตอนที่ 11 — ค่าส่งตามโซน GPS

ระบบคำนวณค่าส่งอัตโนมัติตามระยะทางจาก GPS — โกดังตั้ง Zone ร้านค้าดูได้ใน App

🌐 Admin Web — ตั้งค่า Zone ค่าส่ง

หน้าค่าส่ง — กำหนด Zone ตามระยะทาง (กม.) และราคาค่าส่งแต่ละ Zone
วิธีตั้งค่า:
1. กำหนดระยะทางขอบเขตแต่ละ Zone เช่น Zone A: 0-5 กม. Zone B: 5-15 กม. Zone C: 15+ กม.
2. ตั้งราคาค่าส่งของแต่ละ Zone
3. ระบบคำนวณค่าส่งอัตโนมัติเมื่อลูกค้าสั่งซื้อ

📱 App — ดูตำแหน่งโกดังและ Zone ค่าส่งของร้านคุณ

หน้า GPS — แสดงตำแหน่งโกดัง ระยะทาง และ Zone ค่าส่งของร้านคุณบนแผนที่
หมายเหตุ: ต้องอนุญาตการเข้าถึง GPS บนมือถือ เพื่อให้ระบบวัดระยะทางและคำนวณค่าส่งได้ถูกต้อง

ขั้นตอนที่ 12 — แชท

สื่อสารกับทีมโกดังโดยตรงผ่านระบบแชทในตัว — ทั้ง Admin Web และ App มือถือ

🌐 Admin Web — ระบบแชท

หน้าแชทใน Admin Web — รายการห้องสนทนา สามารถพูดคุยกับโกดังและร้านค้าต่างๆ

📱 App — แชทมือถือ

แท็บ "แชท" ใน App — ส่งข้อความและรูปภาพถึงโกดังได้ทันที
ใช้แชทเพื่อ: สอบถามสินค้า สต็อกและราคา แจ้งปัญหาออเดอร์ หรือขอข้อมูลเพิ่มเติมจากทีมโกดัง

ขั้นตอนที่ 13 — ข่าวสาร / ดีล (Broadcast)

โกดังส่งประกาศ ข่าวสาร และดีลพิเศษมาให้ร้านค้าทุกร้านพร้อมกัน

🌐 Admin Web — ส่ง Broadcast ข่าวสาร

หน้า "แจ้งเตือน" ใน Admin Web — สร้างและส่งข่าวสาร/ดีลไปยังร้านค้าทุกร้านพร้อมกัน

📱 App — รับข่าวสารและดีล

แท็บ "โปรไฟล์" → "ข่าวสาร/ดีล" — รับดีลพิเศษและประกาศจากโกดัง
เปิดการแจ้งเตือน เพื่อรับข่าวสารดีลใหม่ทันที — ตั้งค่าได้ที่ โปรไฟล์ → การแจ้งเตือน → เปิด "โปรโมชั่นและส่วนลด"

ขั้นตอนที่ 14 — ตรวจสอบ Review & Customer Point

ตรวจสอบและอนุมัติรีวิวจากลูกค้า — เมื่ออนุมัติแล้ว ลูกค้าจะได้รับ Customer Point อัตโนมัติ

🌐 Admin Web — รายการ Review

เมนู "ตรวจสอบ REVIEW" — รายการรีวิวจากลูกค้าทั้งหมด พร้อมสถานะอนุมัติ

⭐ รายละเอียด Review และการอนุมัติ

รายละเอียดรีวิว — คะแนนดาว ความคิดเห็น รูปภาพ และข้อมูลลูกค้า
อนุมัติ/ปฏิเสธรีวิว — เมื่ออนุมัติ ลูกค้าจะได้รับ Customer Point ทันที

🌐 Admin Web — Customer Point

เมนู "CUSTOMER POINT" — รายการแต้มสะสมของลูกค้าแต่ละราย
รายละเอียดแต้ม — ประวัติการได้รับและใช้แต้มของลูกค้า
ระบบ Customer Point:
1. ลูกค้าเขียนรีวิวหลังได้รับสินค้า
2. ร้านค้า/โกดังอนุมัติรีวิวใน Admin Web
3. ลูกค้าได้รับแต้มอัตโนมัติ → นำมาแลกส่วนลดครั้งต่อไป

ขั้นตอนที่ 15 — โปรไฟล์ & ตั้งค่า

จัดการข้อมูลส่วนตัว แพ็กเกจ การแจ้งเตือน ข้อมูลร้าน รายงาน และออกจากระบบ

📱 App — หน้าโปรไฟล์หลักและแพ็กเกจ

หน้าโปรไฟล์ — ชื่อผู้ใช้ role แพ็กเกจปัจจุบัน และเมนูตั้งค่าต่างๆ
เลือกแผนการใช้งาน — Free Trial (ฟรี) / Starter ฿299/เดือน / แพ็กเกจอื่นๆ กด "อัปเกรด"
แพ็กเกจ Free Trial: ออเดอร์สูงสุด 50 รายการ ประวัติ 30 วัน
แพ็กเกจ Starter: ออเดอร์ไม่จำกัด ประวัติ 1 ปี Export Excel/PDF รายงานซื้อ-ขาย

👤 แก้ไขโปรไฟล์และตั้งค่าการแจ้งเตือน

แก้ไขโปรไฟล์ — กรอกชื่อและนามสกุล กด "บันทึก"
การแจ้งเตือน — เปิด/ปิด: สถานะคำสั่งซื้อ และโปรโมชั่น/ส่วนลด

🏪 ข้อมูลร้านค้าและการออกจากระบบ

ข้อมูลร้านค้า — ชื่อร้าน เวลาทำการ และช่องทางติดต่อ
ออกจากระบบ — กด "ออกจากระบบ" → ยืนยัน เพื่อ Logout อย่างปลอดภัย

📱 App — รายงานซื้อ-ขาย และข้อมูลร้าน

รายงานซื้อ-ขาย — ยอดขาย รายได้รายวัน/รายเดือน กราฟและสถิติ
ข้อมูลร้านค้า — ชื่อร้าน ที่อยู่ และรายละเอียดการติดต่อ

🌐 Admin Web — ตั้งค่าระบบ

เมนู "ตั้งค่า" ใน Admin Web — ตั้งค่าข้อมูลร้าน นโยบาย และการแจ้งเตือนระบบ
